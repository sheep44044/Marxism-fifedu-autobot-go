# 马克思自动刷题工具

自动完成马克思主义理论课程在线练习题的刷题工具。

## 🚀 快速开始

## 方法一：下载程序直接运行

### 1. 下载程序

下载最新版本的 `main.exe` 和 `config.yaml` 文件，将它们放在**同一个文件夹**下。

### 2. 配置 Cookie

用文本编辑器（如记事本）打开 `config.yaml` 文件，找到 `cookie:` 这一行，将你的 Cookie 粘贴到引号里面。

### 如何获取 Cookie

1. 在浏览器中登录相关学习平台。
2. 按 `F12` 打开开发者工具，切换到 **Network (网络)** 标签。
3. 在网页上随便进行一次点击操作，在网络请求的 `Request Headers` 中找到 `Cookie:` 字段。
4. 复制 `Cookie` 后面的**全部完整内容**（包含 `SESSION=...` 等）。

### 3. 运行

双击 `main.exe` 运行程序即可。

### 📁 文件说明

- `main.exe` - 程序主文件
- `config.yaml` - 配置文件（包含 Cookie 和课程 ID）

---

## 方法二：使用 GoLand等 IDE 直接运行

如果你已经安装 Go 开发环境，或者希望在 IDE 中直接调试、修改代码，可以使用这种方法。

### 1. 配置 Cookie

打开 `config.yaml` 文件，找到 `cookie:` 这一行，将你的 Cookie 粘贴到引号里面。

### 2. 运行

点击 `run` 运行程序即可
