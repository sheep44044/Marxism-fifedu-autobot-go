### 获取 Cookie
1. 在浏览器中登录相关学习平台。
2. 按 `command+option+I/F12` 打开开发者工具，切换到 **Network (网络)** 标签。
3. 在网页上随便进行一次点击操作，在网络请求的 `Request Headers` 中找到 `Cookie:` 字段。
4. 复制 `Cookie` 后面的**全部完整内容**（包含 `SESSION=...` 等）。

### 配置
找到代码开头的配置区，将你复制的内容替换掉 `"YOUR_COOKIE_HERE"`

### 运行
最后运行即可
